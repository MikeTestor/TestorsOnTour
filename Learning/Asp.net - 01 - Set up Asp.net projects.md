# Set up the Asp.net projects

When working with VS Code you set up the projects from the CLI. In Visual Studio you can set up the projects directly in the IDE.

In the CLI:
<pre>
* Create the directory
* Create the solution   : dotnet new sln
* Create the projects   : dotnet new webapi -n TestorsOnTour.API --controllers
* Add projects to sln   : dotnet add TestorsOnTour.API
</pre>

Open VS code with: <pre>*code .*</pre>
